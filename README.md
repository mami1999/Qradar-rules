# SIEM Rules - QRadar Detection Rules Repository

Production-grade detection rules for IBM QRadar SIEM, managed as code with GitOps workflow. All rules are deployed via REST API automation — no manual rule creation in the QRadar console.

## Overview

This repository serves as the single source of truth for QRadar detection content. Rules are written as structured JSON files, peer-reviewed via pull requests, and automatically deployed to QRadar through a CI/CD pipeline.
